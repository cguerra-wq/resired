# ResiRed - Módulo Detector de Huracanes (MVP)

Este repositorio contiene la función Serverless (Python) encargada de consultar el Centro Nacional de Huracanes (NHC) y la plataforma GDACS para identificar amenazas climáticas en tiempo real para San Andrés y Providencia.

## Tecnologías
* Python 3.x
* Azure Functions (Planeado)
* Integración de APIs REST

## Propósito
Actuar como el "gatillo" inicial para el ecosistema de Acción Anticipatoria humanitaria.
